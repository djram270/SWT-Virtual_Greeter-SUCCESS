# Project Start-up Checklist
The following is a set of tasks that you should do once you have your own project repository.

1. Update the [README](./README.md) with the project-specific details.
    * Title and Description.
    * Remaining sections should be populated as the project matures.
2. Fill in the [Participants](./Participants.json) file.
3. Supervisors should have admin privileges on the repo.
4. Create a suitable .gitignore file. Either write it yourselves or find a suitable one from [gitignore repo](https://github.com/github/gitignore/)
5. After these steps, you should be ready to start working on the project.
